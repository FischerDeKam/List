# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Netlify | :x:                |

## Reporting a Vulnerability

To report a vulerability or security issue, please go to the __Issues__ tab and submit a __Security Issue__.

## Data

All list data is private and saved in the browser's local storage.
